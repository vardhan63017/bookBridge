# BookBridge Static Website

This project is now a plain HTML/CSS/JavaScript website.

## Run locally

- Open `index.html` in your browser directly, or
- Install dependencies and start a local server:

```bash
npm install
npm start
```

## Project structure

- `index.html` — home page
- `buy-books.html` — buy page
- `sell-book.html` — sell page
- `categories.html` — categories page
- `contact.html` — contact page
- `book-details.html` — individual book detail page
- `admin.html` — admin approval dashboard
- `css/style.css` — site styles
- `js/` — plain JavaScript logic for pages

## Notes

- No TypeScript or React is used.
- Bootstrap is loaded from CDN for layout simplicity.
- The site is a static front-end project.
